# ユーザーガイド

## 🚀 はじめに

Google Drive Image Copy拡張機能へようこそ！この拡張機能を使用すると、Google Drive内の画像を簡単にクリップボードにコピーして、他のアプリケーションで使用できます。

## 📥 インストール

### Chrome ウェブストアから（公開後）

1. [Chrome ウェブストア](https://chrome.google.com/webstore) にアクセス
2. 「Google Drive Image Copy」を検索
3. 「Chromeに追加」をクリック
4. 確認ダイアログで「拡張機能を追加」をクリック

### 開発版のインストール

1. [GitHub リポジトリ](https://github.com/your-repo) からダウンロード
2. `npm install && npm run build` でビルド
3. `chrome://extensions/` を開く
4. 「デベロッパーモード」をオン
5. 「パッケージ化されていない拡張機能を読み込む」でフォルダを選択

## 🖱️ 基本的な使い方

### 方法1: 右クリックメニュー（推奨）

1. **Google Drive を開く**
   ```
   https://drive.google.com にアクセス
   ```

2. **画像ファイルを表示**
   - 画像ファイルをクリックしてプレビュー表示
   - またはダブルクリックで全画面表示

3. **右クリックでコピー**
   - 画像上で右クリック
   - カスタムメニューから「画像をコピー」を選択
   - 緑のチェックマーク通知で成功確認

4. **他のアプリで貼り付け**
   - Word, PowerPoint, Slack, Discordなどで
   - `Ctrl+V` (Windows) または `Cmd+V` (Mac) で貼り付け


## ⚠️ 重要な制限事項

### Google Driveの共有設定について

**この拡張機能は、画像ファイルが適切な共有権限に設定されている場合のみ動作します。**

#### 必要な共有設定
- ✅ **「リンクを知っている全員が閲覧可」** - 推奨設定
- ✅ **「一般公開」** - 完全公開設定
- ❌ **「制限付き」** - 拡張機能ではコピー不可
- ❌ **特定ユーザーのみ共有** - 拡張機能ではコピー不可

#### 共有設定の変更方法

```
手順:
1. Google Driveで画像ファイルを右クリック
2. 「共有」をクリック
3. 「制限付き」をクリック
4. 「リンクを知っている全員」を選択
5. 「閲覧者」権限を選択
6. 「完了」をクリック
```

#### 権限設定の確認方法

画像ファイルを選択した時に、ファイル名の下に表示されるアイコンで確認できます：
- 🔗 リンクアイコン: 「リンクを知っている全員」設定
- 🌍 地球アイコン: 「一般公開」設定
- 🔒 鍵アイコン: 「制限付き」設定（コピー不可）

## 🎯 対応している画像形式

| 形式 | 対応状況 | 備考 |
|------|---------|------|
| JPEG | ✅ 完全対応 | 自動的にPNGに変換 |
| PNG | ✅ 完全対応 | そのままコピー |
| GIF | ⚠️ 静止画のみ | アニメーションは最初のフレーム |
| WebP | ✅ 対応 | ブラウザ対応に依存 |
| SVG | ❌ 未対応 | 今後対応予定 |

## 🔧 設定とカスタマイズ

### 拡張機能の設定

1. **拡張機能アイコンを右クリック**
2. **「オプション」を選択**（今後実装予定）

現在の設定項目：
- 自動コピー ON/OFF
- 画像品質設定
- 通知表示の ON/OFF

### ブラウザ設定の確認

**クリップボード権限の確認:**
1. `chrome://settings/content/clipboard` にアクセス
2. Google Drive (`https://drive.google.com`) が「許可」になっているか確認
3. 「ブロック」の場合は「許可」に変更

## 🚨 注意事項・制限事項

### 著作権について

```
⚠️ 重要な注意点:
- **共有権限**: 拡張機能を使用するためには「リンク共有」設定が必要
- **著作権**: コピーした画像の著作権は元の所有者に帰属
- **商用利用**: 権利関係を必ず確認してから使用
- **組織ポリシー**: 社内資料は利用規約に従って使用
- **配布**: 無断転載・配布は避ける
```

### 技術的制限

```
制限事項:
1. 画像サイズ: 推奨10MB以下（大きすぎると処理に時間）
2. ネットワーク: インターネット接続が必要
3. 権限: プライベート設定の画像は「リンク共有」に変更が必要
4. ブラウザ: 古いバージョンでは動作しない可能性
```

### プライバシーについて

```
プライバシー保護:
✅ 画像データはローカル処理のみ
✅ サーバーへのデータ送信なし
✅ 閲覧履歴の収集なし
✅ ユーザー情報の保存なし

詳細は Privacy Policy を参照（今後作成）
```

## 🆘 困ったときは

### よくある質問

**Q: 画像がコピーされません**
```
A: 以下を確認してください:
1. クリップボード権限の設定
2. ブラウザの対応状況
3. Google Driveへのログイン状態
詳細は troubleshooting.md を参照
```

**Q: 一部の画像だけコピーできません**
```
A: 画像の権限設定が原因の可能性があります:
1. 画像が非公開設定
2. 組織の制限ポリシー
3. 画像形式の対応状況
手動コピーをお試しください
```

**Q: 画像の画質が落ちます**
```
A: 以下の原因が考えられます:
1. JPEG→PNG変換による軽微な変化
2. ブラウザの画像処理による影響
元画質はほぼ維持されます（95%以上）
```
