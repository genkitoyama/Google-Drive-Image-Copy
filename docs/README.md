# Google Drive Image Copy Extension

Google Driveの画像を右クリックでクリップボードにコピーするChrome拡張機能

## 概要

この拡張機能は、Google Drive内の画像を簡単にクリップボードにコピーできるようにします。通常、Google Driveの画像は直接コピーが困難ですが、この拡張機能により右クリックメニューから簡単にコピーできるようになります。

## 主な機能

- 🖼️ **ワンクリックコピー**: 画像を右クリックするだけでクリップボードにコピー
- 🔄 **自動フォーマット変換**: JPEG画像を自動的にPNG形式に変換
- 🌐 **CORS回避**: Google Driveの制限を技術的に回避
- ⚡ **高速処理**: 最適化されたURL試行により高速コピー
- 🛡️ **エラーハンドリング**: 失敗時の手動コピー指示
- 📂 **スマート判定**: 画像ファイルのみにメニュー表示（Docs/Sheets/PDF等は自動除外）

## インストール

1. このリポジトリをクローンまたはダウンロード
2. `npm install` で依存関係をインストール
3. `npm run build` でビルド
4. Chrome拡張機能ページ (`chrome://extensions/`) を開く
5. デベロッパーモードを有効化
6. 「パッケージ化されていない拡張機能を読み込む」でプロジェクトフォルダを選択

## 使用方法

### 基本的な使い方

1. Google Drive (https://drive.google.com) を開く
2. 画像ファイルをクリックして表示
3. 画像を右クリック
4. カスタムメニューから「画像をコピー」を選択
5. 他のアプリケーション（Word、Slack、画像エディタなど）で Ctrl+V / Cmd+V で貼り付け

### キーボードショートカット

- **Ctrl+Shift+C** (Windows/Linux)
- **Cmd+Shift+C** (Mac)

画像をクリックしてからショートカットキーを押すことでもコピーできます。

### 拡張機能ポップアップ

右上の拡張機能アイコンをクリックすると、ポップアップが表示され、手動でコピー操作を実行できます。
